# BudgetApp

O BudgetApp é uma aplicação simples de controle de orçamento que permite aos usuários adicionar categorias de orçamento. Este README fornece instruções detalhadas sobre como configurar e executar o aplicativo em seu computador pessoal.

## Pré-requisitos

Antes de iniciar, certifique-se de que você tenha as seguintes ferramentas e tecnologias instaladas em seu sistema:

- Xcode: O BudgetApp é desenvolvido em Swift e é executado em um ambiente iOS. Você pode obtê-lo na App Store ou no [site oficial da Apple](https://developer.apple.com/xcode/).

## Configuração do Projeto

Siga as etapas abaixo para configurar o projeto e executá-lo no simulador iOS:

1. Clone o repositório do GitHub para o seu computador pessoal:

```bash
git clone https://github.com/mvinisilva/ProjetoPraticaProfissionalADS
```

2. Abra o projeto no Xcode:

   - Abra o Xcode.
   - Clique em "File" (Arquivo) no menu superior.
   - Selecione "Open" (Abrir).
   - Navegue até o diretório onde você clonou o repositório do BudgetApp e selecione o arquivo "BudgetApp.xcodeproj" para abrir o projeto.

3. Selecione o simulador iOS:

   - No canto superior esquerdo da janela do Xcode, selecione um simulador iOS (por exemplo, iPhone 11) para executar o aplicativo. Se você não tiver simuladores instalados, o Xcode pode oferecer a opção de baixar um.

4. Execute o aplicativo:

   - Clique no botão "Build and Run" (Compilar e Executar) na barra de ferramentas superior do Xcode (ícone de triângulo).

5. O aplicativo será compilado e executado no simulador selecionado.

## Uso do Aplicativo

Após a inicialização do aplicativo no simulador, você poderá usar a interface simples para adicionar categorias de orçamento. Certifique-se de preencher o nome da categoria e o valor do orçamento antes de pressionar o botão "Salvar". Caso contrário, o aplicativo exibirá uma mensagem de erro.
